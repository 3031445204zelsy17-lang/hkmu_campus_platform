**2. 小程序 uploadFile 合法域名**
- `wx.uploadFile`（头像 / 帖子 / 失物配图）必须在**微信公众平台 → 开发管理 → 服务器域名 → uploadFile 合法域名**配 `https://hkmu-campus-sea.azurewebsites.net`。
- 开发者工具 `urlCheck: false` 会掩盖此问题——**本地测过 ≠ 真机/体验版过**。没配则真机上传全挂。
- 待办：登录公众平台确认该项已配置。

**3. refresh 竞态（已修代码，待部署）**
- 现象：access_token 2h 过期后，app 启动时 `onLaunch` + `home.onShow` 并发恢复 session → 并发 refresh → 后端 refresh token rotation → 第二个失败 → `clearSession` 把全局 session 搞坏 → 用户被踢、持续 401。
- 已在 `request.js` 加 `_refreshPromise` 单例互斥（本地）。
- 待办：commit + 部署后才在生产生效。
